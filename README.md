## 运行方式：
### 搭建运行环境:
```shell
git clone https://github.com/dailin3/NMSTest.git
cd nmstest
pip install -r requirement.txt
```
### 初级:
#### 运行服务器
```shell
./NMS
```
#### 运行客户端
```shell
python NMS_junior_3.py
```
### 高级:
#### 运行服务器
```shell
./NMS -a
```
#### 运行客户端
```shell
python NMS_senior_3.py
```
